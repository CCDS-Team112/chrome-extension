# Demo Script (2 minutes)

## Setup
1. Load the extension from `apps/extension`.
2. Open a stable site (example: a shopping cart page or a simple form page).

## Flow
1. Baseline:
   - Use the mouse to find and click a small control or input a value.
2. With A11y Autopilot:
   - Press `Ctrl+Shift+P` to open the palette.
   - Say or type: `scroll down`
   - Type: `click checkout`
   - Type: `type test@example.com into email`
   - Turn on labels: `label mode on`
   - Say/type: `open 3`

## Metrics
Read the demo metrics line in the palette:
- Steps
- Time elapsed
- Ambiguity prompts

## Notes
If ambiguous matches appear, select a number 1-9 from the list.
